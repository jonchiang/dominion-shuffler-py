Dominon Shuffler
================

Use the script to setup a new game of Dominion.

	python dominion-shuffle

Sample output
-------------

1. cellar
2. chancellor
3. chapel
4. council-room
5. gardens
6. laboratory
7. library
8. money-lender
9. witch
10. workshop

Randomize from multiple sets.

	python dominion-shuffle --names 'dominion,prosperity'

Sample output
-------------

From prosperity:
1. Hoard
2. Mountebank
3. Trade Route
4. Venture
5. Worker's Village

From dominion:
1. Adventurer
2. Chapel
3. Gardens
4. Thief
5. Village
