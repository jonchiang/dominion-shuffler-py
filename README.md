Dominon Shuffler
================

Use the script to setup a new game of Dominion.

	python dominion-shuffle

Sample output

	1. Cellar
	2. Chancellor
	3. Chapel
	4. Council-room
	5. Gardens
	6. Laboratory
	7. Library
	8. Money-lender
	9. Witch
	10. Workshop

-------------
Randomize from multiple sets.

	python dominion-shuffle --names 'dominion,prosperity'

Sample output

	From prosperity:
	1. Hoard
	2. Mountebank
	3. Trade Route
	4. Venture
	5. Worker's Village
	
	From dominion:
	1. Adventurer
	2. Chapel
	3. Gardens
	4. Thief
	5. Village
